FIGMA AI PROMPT: Product Deal Scraper & Price Comparison Platform
================================================================================

PHASE 1: COMPLETE APP OVERVIEW

================================================================================
App Type & Purpose:
================================================================================
A multi-category product deal aggregation and price comparison platform that 
scrapes real-time pricing data from multiple retailers. Users can browse 
products across categories (phones, laptops, cosmetics, shoes, sound systems), 
compare prices, set price alerts, track favorites, and get the best deals in 
their region.

================================================================================
Core Features (5-7 Main Functionalities):
================================================================================
1. Product Browse & Discovery - Category-based browsing with search functionality
2. Price Comparison - View same product from multiple retailers with price differences
3. Price Alerts - Set notifications when product prices drop below user threshold
4. Favorites Management - Save and organize favorite products for later
5. Real-time Deals - Daily featured deals with discount percentages
6. Product Details - Comprehensive product information with specs, reviews, retailer links
7. User Authentication - Login/signup for personalized features (favorites, alerts, plans)

================================================================================
User Flow:
================================================================================
User lands → Browse categories OR Search → View products/deals → Compare prices 
→ Save to favorites OR Set price alert → Proceed to purchase via retailer link 
→ Track alerts/favorites in account

================================================================================
Page Structures (Distinct Pages/Screens):
================================================================================
1. Home/Main Page - Featured deals showcase + category discovery cards
2. Category Page - Products grid filtered by category with sorting/filtering
3. Product Details Page - Single product view with price history, retailer list, specs
4. Search Results Page - Dynamic results from search query
5. Favorites Page - User's saved products (authenticated users only)
6. Price Alerts Page - Manage active and past price alerts
7. Account/Profile Page - User preferences, subscription plans, settings
8. Authentication Page - Login/signup/register forms

DESIGN SYSTEM SPECIFICATIONS

================================================================================
Color Palette:
================================================================================
- Primary: White (#ffffff) - backgrounds
- Text: Dark Gray (#1a1a1a) - main text
- Accent: Orange (#ff6b00) - CTAs, highlights, borders on hover
- Border: Light Gray (#d8d8d8) - 1px borders
- Header: White background with dark text
- Hover: Light orange background (rgba(255, 107, 0, 0.08))

================================================================================
Typography:
================================================================================
- Font: Inter (system font fallback)
- Headings: 600-700 weight
- Body: 400-500 weight
- Sizes: 14px (small), 16px (body), 20px (subheading), 28px (heading)

================================================================================
Spacing System:
================================================================================
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- Grid gaps: 16px

================================================================================
Component Heights:
================================================================================
- Header: 80px
- Buttons: 44px minimum (touch-friendly)
- Input fields: 40px
- Product cards: Responsive (aspect ratio 1:1 for images)

================================================================================
Responsive Breakpoints:
================================================================================
- Mobile: < 480px
- Tablet: 480px - 768px
- Desktop: 768px - 1200px
- Large Desktop: > 1400px

================================================================================
Accessibility Standards:
================================================================================
- All interactive elements: WCAG AA compliant
- Color contrast: 4.5:1 minimum
- Focus states: Visible 2px outline
- Semantic HTML: Proper heading hierarchy, aria-labels
- Animations: Prefer reduced motion respected

================================================================================

ENGINEERING BEST PRACTICES FOR IMPLEMENTATION

================================================================================
Frontend Architecture:
================================================================================
- Build with vanilla JavaScript (ES6 modules) + CSS Grid/Flexbox
- No component framework (maintain lightweight)
- Semantic HTML structure
- Reusable CSS component library (not image-based)
- Module-based organization (header.js, product-grid.js, etc.)

================================================================================
Performance:
================================================================================
- Lazy load product images
- Pagination instead of infinite scroll where possible
- Cache API responses appropriately
- Minify and bundle CSS/JS

================================================================================
Code Quality:
================================================================================
- Maintain consistent file structure
- Comment complex logic
- Use descriptive class/ID names
- Version control for CSS (query params: ?v=timestamp)

================================================================================
Security:
================================================================================
- Sanitize user input on client-side
- Secure API endpoints (validate on backend)
- Store auth tokens securely
- HTTPS only
